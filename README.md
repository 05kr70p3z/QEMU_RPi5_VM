# QEMU_RPi5_VM
---
A qemu vm emulating a raspberry Pi with 32gb storage and 8GB of ram.

##Goal
---
Emulate raspberry pi 5 specifically from the turbo pi kit reffered [here](https://www.hiwonder.com/products/turbopi?variant=40112905388119)

Intention is to be able to remotely and independently test software / modules developed by my team in our CSUSM Capstone Project.
